# Space-Shooter

## Controls
- Move  : W A S D Keys
- Fire  : Left Mouse Click
- Pause : Escape Key

---

## Linux build

### Dependencies
- C++ >= 17
- SFML  >= 2.5.1
- CMake >= 3.0

```
sudo apt-get install cmake libsfml-dev
```
### Build

1. Clone repository  `git clone https://github.com/Karansemwal/Space-Shooter.git`

2. `cd Space-Shooter`

3. `mkdir build && cd build`

4. `cmake .. && make`

5. Run game  `./Space_Shooter`
